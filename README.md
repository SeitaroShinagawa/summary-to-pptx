# summary-to-pptx

## Develop environment
- Windows
- python-pptx (0.6.10)

## Usage
```
python make_slide.py --config example.config --img_dict images --out out.pptx
```
Note: executable on Windows (an error of python-pptx happens at text_frame.fit_text (editing font) on Linux)
